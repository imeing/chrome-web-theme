# chrome-web-theme

### description

Google chrome web application을 만드는 레포지토리입니다.

기본적인 Vanilla.js를 공부하기 위한 프로젝트이며 새 탭을 열었을 때 다음과 같은 기능을 제공합니다.

- 현재 장소에 따른 날씨 및 시간
- 로컬 스토리지를 이용한 유저 계정 관리
- 일러스트 배경화면
